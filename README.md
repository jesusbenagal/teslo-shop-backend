# TesloAPI

1. Clonar Repo
2. ```npm i || yarn install```
3. Crear archivo .env en base al .env.template
4. Levantar la base de datos

```docker-compose up -d```

5. Ejecutar Seed
```
http://localhost:3000/api/seed
```

6. Levantar en modo desarrollo 

```npm run start:dev```
